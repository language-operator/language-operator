Getting started:

1. Create a LanguageModel to configure your LLM:

   kubectl apply -f - <<EOF
   apiVersion: langop.io/v1alpha1
   kind: LanguageModel
   metadata:
     name: my-model
   spec:
     provider: openai
     modelName: gpt-4
     apiKeySecretRef:
       name: openai-api-key
       key: api-key
   EOF

2. Create a LanguageTool to deploy MCP tools:

   kubectl apply -f - <<EOF
   apiVersion: langop.io/v1alpha1
   kind: LanguageTool
   metadata:
     name: my-tool
   spec:
     image: myregistry/mcp-tool:latest
     type: http
     port: 8080
   EOF

3. Create a LanguageClient for user interaction:

   kubectl apply -f - <<EOF
   apiVersion: langop.io/v1alpha1
   kind: LanguageClient
   metadata:
     name: my-client
   spec:
     image: myregistry/language-client:latest
     type: web
     modelRefs:
     - name: my-model
       role: primary
     toolRefs:
     - name: my-tool
   EOF
